# Campus Life Super App

## Description and Purpose
The campus life super app is a multi-page web app that makes it easy for students to get to important campus information. The goal of this project is to give users a place to find out about campus events, see dining options, and find their way around some useful student resources.

This project was created as part of a web development course to demonstrate an understanding of front-end development concepts so things such as responsive design, JavaScript interactivity, and API integration.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- External API (JSONPlaceholder)

---

## Setup & Deployment
To run this project locally:
1. Download or clone the repository from GitHub
2. Open the project folder
3. Open `index.html` in your web browser

To view the deployed version:
- Open the GitHub Pages link provided in the repository

---

## Features & How to Use
- **Multi-page navigation:** Users can navigate between Home, Events, and Dining pages using the navigation bar
- **Responsive design:** The layout adjusts for desktop, tablet, and mobile screens
- **Dynamic event loading:** Users can click the "Load Events" button on the Events page to fetch and display event data from an external API
- **Interactive buttons:** Quick access buttons allow users to easily navigate between sections
- **Styled UI:** Custom CSS and Bootstrap are used to create a clean and consistent design
